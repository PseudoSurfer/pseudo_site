Starting Again

24 Apr 2021 - Pseudo Surfer

After an accident, I’ve lost access to my hard drive so while I wait to access it, I’m starting again.

My 2014 MacBook Pro died after liquid got spilled on it so I’ve bought myself a new M1 Mac Mini. I’ll get a hard drive enclosure to access the old hard drive but in the meantime I’m recreating my site. I did not save my files to GitHub properly and I guess this little incident will be a good lesson on how to keep all my important stuff together.

Since starting my Jekyll site I’ve added and removed multiple plug-ins and assets. By starting again I can have a fresh install from which to work. For instance I’m using a gallery creator that builds static html gallery pages. The creator of that script uses Bootstrap and so instead of fighting against it I’ve decided to start using Bootstrap from the beginning.
